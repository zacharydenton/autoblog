Version 3.0.4 of WordPress, instantly accessible via the update page in your dashboard or for download here , is a very important update to apply to your sites as soon as possible because it contains a kernel security bug in our HTML sanitation, libraries, called KSES. I would rate this release as "critical".

This issue affects all versions of WordPress before 3.0.4, so if you're still on a 2.x release you need to update also.

I appreciate an update during the holidays is no fun, but this is worth putting down the eggnog. In the spirit of the holidays, consider helping your friends, too.

If you are a security researcher, we appreciate you taking a look at this changeset , and review our updated. We have given it a lot of thought and review, but since this is so basic that we want as many brains in it as possible. Thanks Mauro Gentile and Jon Cave (duck_) who discovered and alerted us to these XSS vulnerabilities first.
